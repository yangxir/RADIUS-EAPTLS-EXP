# このプロジェクトについて

ようこそ、**RADIUS EAP-TLS 実験記録**へ。

## 🔍 プロジェクト概要
本プロジェクトでは、**FreeRADIUS** を使用した **EAP-TLS 認証実験** を記録し、以下の目標を達成することを目指しています。

- **EAP-TLS 認証のセットアップ**
- **プロキシサーバーの追加**
- **認証パフォーマンスの最適化**
- **WiFi 認証のセキュリティ強化**

## 📌 実験フェーズ
1️⃣ **フェーズ 1**：EAP-TLS 認証のセットアップ  
2️⃣ **フェーズ 2**：プロキシサーバーの追加  
3️⃣ **フェーズ 3**：高度な認証最適化（準備中）  
4️⃣ **フェーズ 4**：セキュリティとパフォーマンス強化（準備中）
